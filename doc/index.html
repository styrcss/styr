Install
% npm install styr --save

See dst directory. styr.css and styr.min.css is available.
Or, you can import only some styles as you need from style files in src, directly. (Styr uses stylus as syntax).

Build
# to build, you need to install dependencies
% npm install

# then
% npm run build
Policy

Styr does not contain...

  • JavaScript
  • CSS Animation
  • Web Font
  • Image (besides base64 embeded one in css)
